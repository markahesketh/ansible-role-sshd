# Ansible Role: sshd

Ansible role to manage sshd.

## Installation

```
ansible-galaxy install markahesketh.sshd
```

## Role Variables

Default values are listed below (see [`defaults/main.yml`](defaults/main.yml)):

```yml
sshd_permit_root_login: no
sshd_password_authentication: no
```
